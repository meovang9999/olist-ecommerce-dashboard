# olist-ecommerce-dashboard
The main goal of this project is to practice and improve my basic Power BI skills. The dashboard analyzes sales performance, customer behavior, product insights, and delivery performance to generate meaningful business insights.
All data modeling, cleaning, and calculations are handled directly in Power BI. DAX measures are used to calculate KPIs. The dashboard is designed to be interactive, allowing users to filter by date, state, product category, city.

Dataset: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
